Aura Transpiler - Phase 1

This workspace contains the initial scaffolding for the Aura language transpiler.

Phase 1 objectives:
- ANTLR4 grammar (parser/aura.g4)
- AST node classes (transpiler/ast.py)
- Simple transformer (transpiler/transformer.py)
- Project layout and CLI entry (main.py)

Next steps:
- Generate parser classes with ANTLR4
- Implement parser -> AST conversion
- Add tests and CI
