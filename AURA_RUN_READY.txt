BASIC USAGE

  python3 main.py run examples/hello.aura
  python3 main.py run examples/fibonacci.aura
  ./aura run examples/prime_checker.aura

WITH VERBOSE (see generated Python code):

  python3 main.py run examples/hello.aura -v
